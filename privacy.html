Privacy Policy

Program Name: Adam's Game SMS Program

Adam's Game SMS Program sends low-volume, opt-in automated SMS notifications for an event-based social game. Participants may receive game start messages, participation confirmation messages, and automated challenge notifications while a game session is active.

Information Collected

The program may collect participant names, mobile phone numbers, game participation status, and basic message responses such as OPTIN, OPTOUT, HELP, and STOP.

How Information Is Used

Participant information is used only to operate the SMS game program, send game start messages, process participation responses, send challenge notifications to opted-in participants, respond to HELP requests, and process STOP removal requests.

Data Sharing

Mobile phone numbers, SMS consent, and opt-in data will not be shared, sold, rented, or disclosed to third parties or affiliates for marketing or promotional purposes.

Opt-Out

Participants may reply STOP at any time to be permanently removed from the player list. Participants may reply HELP for help.

Contact

For questions about this SMS program, contact the organizer directly.
